競馬の歴史

伝説は、いつも時代を超えて語り継がれる。
日本競馬を震わせた名馬たちの軌跡を、年代順に辿る。

昭和の伝説
1984年
世界への挑戦
カツラギエース、日本馬初のジャパンカップ制覇
1984年 ジャパンカップ

外国馬が圧倒的優位とされた第4回ジャパンカップで、カツラギエースが逃げ切り勝ち。日本競馬の底力を世界に示した歴史的勝利。

1981年に創設されたジャパンカップは、世界の強豪馬を日本に招いて覇を競う国際招待競走だった。しかし創設から3年間、日本馬は一度も外国馬に勝てず、「日本馬では歯が立たない」という空気が漂っていた。

1984年、その流れを断ち切ったのが西浦勝一騎手とカツラギエースだった。逃げの手に打って出たカツラギエースは、世界の強豪がひしめく中、ハナを奪ってそのままゴールまで駆け抜けた。

「日本の馬もついに世界に通じる日が来た」——当時の競馬関係者の証言

この勝利は単なるレース結果にとどまらず、日本競馬が世界と互角に戦えるという証明となり、その後の日本競馬の国際化を加速させる大きな転換点となった。

📋 1984年 ジャパンカップ 1着 / 鞍上：西浦勝一
1984年
無敗の三冠
皇帝シンボリルドルフ、史上初の無敗三冠達成
1984年 皐月賞・日本ダービー・菊花賞

「皇帝」の異名を持つシンボリルドルフが史上初の無敗三冠を達成。圧倒的な強さで日本競馬史に君臨し続けた。

シンボリルドルフが史上初の無敗での三冠を達成したのは1984年。鞍上の岡部幸雄騎手とのコンビで皐月賞・日本ダービー・菊花賞を完全制覇した。

「あの馬には知性があった。レースを理解して走っていた」——岡部幸雄騎手
📋 1984年 三冠達成 / 通算16戦13勝 / 鞍上：岡部幸雄
1990年
奇跡の復活
オグリキャップ、引退レースで奇跡の復活優勝
1990年 有馬記念

不振が続き引退を発表した後の有馬記念。「もう終わった」と誰もが思った芦毛の怪物が、武豊を背に奇跡の復活優勝。17万人超の大歓声は今も伝説として語り継がれる。

1990年の有馬記念。オグリキャップはその年春から不振が続き、秋のジャパンカップでも11着と大敗していた。

「オグリ！オグリ！オグリ！」——1990年有馬記念、中山競馬場を包んだ歓声
📋 1990年 有馬記念 1着 / 鞍上：武豊 / 観客数17万人超
1991年
大番狂わせ
ダイユウサク、13番人気で有馬記念を大金星
1991年 有馬記念

圧倒的1番人気のメジロマックイーンが揃った有馬記念で、13番人気のダイユウサクが大逃げを打って大金星。「競馬に絶対はない」を体現した歴史的番狂わせ。

1991年の有馬記念は、天皇賞春秋連覇の絶対王者・メジロマックイーンが圧倒的1番人気に推され、誰もがメジロマックイーンの勝利を疑わなかった。

「信じられない。競馬とはこういうものだ」——当時の実況より
📋 1991年 有馬記念 1着（13番人気） / 鞍上：熊沢重文
1999年
空前絶後の末脚
ブロードアピール、根岸Sで歴史に残る大外一気
1999年 根岸ステークス

根岸ステークスで見せた後方最後方からの信じられない末脚での差し切り勝ち。そのインパクトは今も「競馬史上最高の末脚の一つ」として語り継がれる。

「あんな競馬は見たことがない。あれは末脚ではなく、もはや別の何かだ」——当時の競馬評論家
📋 1999年 根岸ステークス 1着 / 鞍上：福永祐一
2000年
世紀末覇王
テイエムオペラオー、古馬G1を完全制覇し年間無敗
2000年 古馬G1完全制覇

2000年、古馬の主要G1を全て制覇し年間無敗を達成した「世紀末覇王」。前人未到の古馬G1完全制覇は永遠に語り継がれる。

「2000年の競馬はオペラオーのためにあった」——和田竜二騎手
📋 2000年 古馬G1全勝（5戦5勝） / 年間獲得賞金歴代1位 / 鞍上：和田竜二
平成の黄金時代
2005年
無敗の三冠
ディープインパクト、無敗の三冠達成
2005年 皐月賞・日本ダービー・菊花賞

「日本近代競馬の結晶」と称された名馬が、武豊を背に飛ぶような走りで史上2頭目の無敗三冠を達成した。

「走っているというより、飛んでいる感覚だった」——武豊騎手
📋 2005年 無敗三冠達成 / 通算14戦12勝 / 鞍上：武豊
2012・2013年
世界最高峰
オルフェーヴル、凱旋門賞2年連続2着。悲願あと一歩
2012年・2013年 凱旋門賞

日本馬の悲願・凱旋門賞制覇に最も近づいた一頭。2012年は残り200mで先頭に立ちながら差し返されてハナ差2着。翌年も2着と、日本競馬を世界最高峰へと近づけた。

「あの200mのことは、一生忘れられない」——池添謙一騎手
📋 2012年・2013年 凱旋門賞 ともに2着 / 鞍上：池添謙一
2013年
父子の絆
キズナ、ダービー制覇。父ディープと親子ダービー馬に
2013年 日本ダービー

父ディープインパクトに続き日本ダービーを制したキズナ。「日本を繋ぐ」という名の通り、父子二代でのダービー制覇という感動的なドラマを届けた。

「ディープの仔でダービーを勝てた。あの馬への恩返しができた気がした」——武豊騎手
📋 2013年 日本ダービー 1着 / 鞍上：武豊 / 父ディープとの親子ダービー制覇
2017年
悲願の戴冠
シュヴァルグラン、悲願のジャパンカップ制覇
2017年 ジャパンカップ

幾度もビッグレースで惜敗を繰り返してきたシュヴァルグランがついにジャパンカップを制覇。遅咲きの大器が掴んだ悲願の頂点。

📋 2017年 ジャパンカップ 1着 / 鞍上：H・ボウマン
2018年
世界レコード
アーモンドアイ世界レコード。そしてキセキの大逃げ
2018年 ジャパンカップ

3歳牝馬ながら2分20秒6の世界レコードで圧勝。そのレースでキセキが見せた世界的な大逃げもまた伝説——あの日は二つの伝説が生まれた。

📋 2018年 ジャパンカップ / アーモンドアイ1着・世界レコード2分20秒6 / キセキ3着
令和の新時代
2020年
史上初・同年無敗三冠
コントレイル×デアリングタクト、史上初の同年無敗三冠同時誕生
2020年 菊花賞・秋華賞

父ディープの血を継いだコントレイルが無敗三冠達成。同年デアリングタクトも無敗の牝馬三冠を達成し、史上初の「同年無敗三冠同時誕生」という奇跡が起きた。

📋 コントレイル：2020年 牡馬三冠（無敗） / デアリングタクト：2020年 牝馬三冠（無敗）
2020年
芝G1・9勝
アーモンドアイ、芝G1・9勝の日本記録を樹立して引退
2020年 天皇賞（秋）・有馬記念

天皇賞（秋）でG1最多勝記録を更新し、最終的に芝G1・9勝という空前絶後の記録を打ち立てて引退。「史上最強牝馬」の称号は永遠に輝き続ける。

「この馬に出会えた競馬人生は幸せだった」——ルメール騎手
📋 通算芝G1・9勝（日本記録） / 鞍上：ルメール（主戦） / 2020年引退
2022年
最速ダービー馬
ドウデュース、最速タイのレコードでダービー制覇
2022年 日本ダービー

武豊騎手とのコンビで最速タイのレコードタイムでダービーを制覇。「世界を駆ける末脚」でその後も国内外を問わず頂点を目指し続けた。

📋 2022年 日本ダービー 1着・レコードタイム / 鞍上：武豊（ダービー6勝目）
2023年
世界ランキング1位
イクイノックス、世界ランキング1位に輝く
2023年 ジャパンカップ・世界1位認定

ジャパンカップを圧勝し、国際競馬統括機関連盟が発表する世界ランキングで日本馬として初の1位を獲得。「天衣無縫」の走りで世界の頂点に立った。

📋 2023年 世界ランキング1位（日本馬初） / 通算12戦10勝 / 鞍上：ルメール
2024・2025年
世界制覇
フォーエバーヤング、BCクラシック制覇＆サウジカップ連覇
2024年 BCクラシック / 2025年 サウジカップ連覇

日本のダート路線から世界へ飛び出し、BCクラシックを制覇。サウジカップを連覇するという偉業も達成。日本ダート馬が世界最高峰の舞台で頂点に立った。

📋 2024年 BCクラシック 1着 / サウジカップ連覇 / 日本ダート馬として世界制覇
